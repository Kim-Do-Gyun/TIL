# 택스트 작성하기
### hn 태그 
제목이나 주제를 나타내는 텍스트를 표현할 때 사용
``` html
<hn>제목<hn>
```
---
``` html
<h1>Heading level</h1>
<h2>Heading level</h2>
<h3>Heading level</h3>
<h4>Heading level</h4>
<h5>Heading level</h5>
<h6>Heading level</h6>
```
---
hn 태그로 작성된 키워드는 검색 엔진에서 키워드로 인식
h1이 가장 중요도가 높으며 크고 굵게 표현 됨
### p 태그
본문의 문단을 작성할 떄 사용
``` html
<p>내용</p>
```
---
``` html
<p>p 태그는 하나의 문단을 작성할 때 사용된다.</p>
```
---
#### br태그
문단에서 줄 바꿈 할 때 사용
``` html
<br>
```
p 태그 안에서 줄바꿈을 하더라도 br 태그를 사용하지 않으면 줄 바꿈이 되지 않은 채 출력됨
