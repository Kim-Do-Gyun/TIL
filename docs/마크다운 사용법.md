# 10. 테이블
```
|으로 구분. - 으로 구분된 곳 왼쪽, 오른쪽, 양쪽에 :을 붙혀 왼쪽,오른쪽, 가운데 정렬이 가능하다.

기본적인 문법사용이 가능하다.

| 월 | 화 | 수 |
| :--- | :---: | ---: |
| 왼쪽 | 가운데 | 오른쪽 |
| **볼드** | ~~취소선~~ | *기울임* |
```

| 월      |    화    |     수 |
| :----- | :-----: | ----: |
| 왼쪽     |   가운데   |   오른쪽 |
| **볼드** | ~~취소선~~ | *기울임* |

# 11. 체크박스
```
-, *, + 등 목록 생성 이후 띄어쓰기, 이후 대괄호입력으로 생성
대괄호 안에 띄어쓰기시 빈 체크박스, x를 넣으면 체크된 박스 생성

- [ ] 빈박스
- [x] 체크 박스
```

- [ ] 빈박스
- [x] 체크 박스
